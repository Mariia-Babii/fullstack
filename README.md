# 🎨 CSS Specificity Assignment

Цей проєкт — це практична лабораторія для вивчення **специфічності CSS**. Він демонструє, як різні типи селекторів впливають на пріоритет стилів та фінальний вигляд елементів у браузері.

## 🌟 Особливості проєкту
- Розрахунок ваги за системою `(Inline, ID, Class, Tag)`.
- Візуалізація рівнів пріоритету за допомогою кольорових блоків.
- Використання комбінаторів, атрибутів та псевдокласів.

## 📊 Таблиця ваги селекторів

| Селектор | Вага | Опис |
| :--- | :--- | :--- |
| `main` | **1** | Селектор тегу |
| `div p` | **2** | Вкладені теги |
| `.simple-box` | **10** | Селектор класу |
| `[data-role="alert"]` | **10** | Селектор атрибута |
| `.content > h1` | **11** | Дочірній елемент |
| `div.card[title="container"]` | **21** | Тег + Клас + Атрибут |
| `#unique-block` | **100** | Селектор ID |
| `#main-footer span.copyright` | **111** | ID + Тег + Клас |
